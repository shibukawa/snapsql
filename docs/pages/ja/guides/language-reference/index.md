### 実装済み

- [Go](./go.md) - Go 言語での使い方

### 計画中

- TypeScript - TypeScript/JavaScript での使い方
- Python - Python での使い方
- Rust - Rust での使い方

## 共通の概念

すべての言語で以下の機能が利用できます：

### 型安全性

生成されたコードは型安全で、コンパイル時にパラメータと戻り値の型チェックが行われます。

### システムカラム

`context` や類似の仕組みを通じてシステムカラム（ユーザーID、タイムスタンプなど）を注入できます。

### トランザクション

データベーストランザクション内でクエリを実行できます。

### モック

テスト時にモックを使用できます（言語により実装状況が異なります）。
